# Attendance Tracking App (Flutter)

A simple Flutter app with three roles:

- **Student**: View attendance and raise objections to increase attendance for events like hackathons.
- **Teacher**: Track daily attendance and review/approve/reject student objections.
- **Admin**: Create and manage credentials for students, teachers, and admins.

## Run

```bash
flutter pub get
flutter run
```

## Install Flutter (if not installed)

```bash
# Linux example
cd /opt
sudo curl -LO https://storage.googleapis.com/flutter_infra_release/releases/stable/linux/flutter_linux_3.24.5-stable.tar.xz
sudo tar -xf flutter_linux_3.24.5-stable.tar.xz
export PATH="$PATH:/opt/flutter/bin"
flutter doctor
```
